## Instalação

### Dependências
Para instalar todas as dependências, utilize o comando:

    npm install
    
Caso esteja ocorrendo algum erro, tente instalar as versões atualizadas:

    bcryptjs
    body-parser
    connect-flash
    cookie-parser
    express
    express-handlebars
    express-session
    mongoose
    passport
    passport-local
